# TeachKaam Software Solutions Website

## Overview

TeachKaam is a modern software solutions company website developed to showcase business services, software products, and digital transformation solutions. The platform provides information about ERP systems, POS solutions, FBR integrations, SCADA systems, web development, mobile app development, and API integrations.

The website is designed with a clean, responsive, and professional user interface to help businesses explore TeachKaam services and contact the company for consultations.

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/teachkaam-website.git
```

Move into project directory:

```bash
cd teachkaam-website
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
